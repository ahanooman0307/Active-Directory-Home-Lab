# Active Directory Home Lab

## Overview
This project documents a virtual Active Directory lab built using VirtualBox. The lab includes a Windows Server Domain Controller, a custom Active Directory domain, automated user creation with PowerShell, and a Windows client joined to the domain.

## Lab Environment
- Virtualization: Oracle VirtualBox
- Server: Windows Server
- Client: Windows 10/11
- Services: Active Directory Domain Services, DNS
- Tools: PowerShell, Active Directory Users and Computers

## Goals
- Build a Windows domain environment
- Configure a Domain Controller
- Create and manage Active Directory users
- Automate user provisioning with PowerShell
- Join a client workstation to the domain
- Test domain authentication using created user accounts

## What I Built
- Installed and configured Windows Server as a Domain Controller
- Created a custom Active Directory domain
- Used PowerShell to generate 1,000 user accounts
- Joined a Windows client VM to the domain
- Logged into the client using a domain user account

## Skills Demonstrated
- Active Directory administration
- Windows Server configuration
- PowerShell automation
- Domain user management
- Client domain joining
- Basic enterprise IT troubleshooting
